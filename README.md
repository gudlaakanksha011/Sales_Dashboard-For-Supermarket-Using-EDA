# Supermarket Sales Exploratory Data Analysis (EDA)

## 📊 Project Overview
This project features an interactive **Supermarket Sales Dashboard** built in Microsoft Excel. The analysis focuses on understanding historical sales trends, customer demographics, and product performance to provide actionable business insights.

## 🚀 Key Features
* **Dynamic Visualizations:** Real-time updates using Excel Slicers for Branch, Date, and Quantity.
* **KPI Tracking:** High-level summary of Total Revenue ($299,899.72), Total Profit, and Customer Ratings.
* **Trend Analysis:** Monthly revenue tracking to identify seasonal peaks and troughs.
* **Customer Segmentation:** Analysis of Gender-wise revenue and Member vs. Normal customer behavior.

## 🔍 Key Insights
* **Gender Contribution:** Female customers drive the majority of sales, accounting for **57%** of total revenue.
* **Product Performance:** Fashion Accessories and Food & Beverages are the highest revenue-generating categories.
* **Payment Preferences:** Clear visualization of E-wallet, Cash, and Credit Card usage across different customer types.
* **Seasonal Trends:** Significant sales activity is concentrated in the early months of the year (Jan-March).

## 🛠️ Tools & Techniques Used
* **Data Cleaning:** Standardizing product categories and handling date formats.
* **Pivot Tables & Pivot Charts:** For complex data aggregation.
* **Dashboard Design:** Using Slicers and Timeline filters for an interactive user experience.
* **Excel Functions:** Implementation of logical and statistical functions for KPI calculations.

## 📂 How to View
1. Download the `.xlsx` file from this repository.
2. Open the file in **Microsoft Excel**.
3. Use the **Slicers** on the right side of the dashboard to filter the data by Branch or Date.
